EngramAI Lite REST API

The EngramAI Lite API allows you to interact with the knowledge graph programmatically. All endpoints return JSON responses.

Base URL

All API endpoints are relative to: http://localhost:3000/api

Authentication

Currently, the API does not require authentication.

Engram Endpoints

GET /engrams

Returns a list of all engrams.

Example Response:

{ "engrams": [ { "id": "01234567-89ab-cdef-0123-456789abcdef", "content": "The application should provide weather forecasts for user-specified locations.", "source": "project_manager", "confidence": 0.95, "timestamp": "2023-05-10T12:00:00Z", "metadata": { "type": "requirement", "priority": "high" } }, ... ] }
GET /engrams/{id}

Returns a specific engram by ID.

Parameters:

Name Type Description
id Required string The unique identifier of the engram.
POST /engrams

Creates a new engram.

Request Body:

Name Type Description
content Required string The knowledge content of the engram.
source Required string The source of the knowledge.
confidence Optional number Confidence score between 0.0 and 1.0. Defaults to 1.0.
metadata Optional object Additional structured metadata for the engram.

Example Request:

{ "content": "React components should follow the single responsibility principle.", "source": "developer", "confidence": 0.9, "metadata": { "type": "design_principle", "area": "frontend" } }
DELETE /engrams/{id}

Deletes an engram by ID.

Parameters:

Name Type Description
id Required string The unique identifier of the engram to delete.

Connection Endpoints

GET /connections

Returns a list of all connections.

POST /connections

Creates a new connection between engrams.

Request Body:

Name Type Description
source_id Required string ID of the source engram.
target_id Required string ID of the target engram.
relationship_type Required string Type of relationship (e.g., "causes", "supports", "contradicts").
weight Optional number Connection strength between 0.0 and 1.0. Defaults to 1.0.

Search Endpoints

POST /search

Search for engrams using various criteria.

Request Body:

Name Type Description
query Optional string Text query for semantic search.
source Optional string Filter by source.
min_confidence Optional number Minimum confidence threshold.
limit Optional integer Maximum number of results to return. Default is 10.

Example Request:

{ "query": "weather API", "source": "developer", "min_confidence": 0.7, "limit": 5 }
