Demo Multi-Agent Coding Project

This dashboard is displaying data from a multi-agent coding collaboration scenario:

  • Software Developer: Designs architecture and implements code
  • Quality Assurance Engineer: Tests code and reports issues
  • Technical Writer: Creates documentation
  • Project Manager: Coordinates the team and tracks requirements

The agents are working together on a weather application project, sharing knowledge through engrams connected in a semantic network.

API Endpoints

GET /api/engrams
List all engrams
GET /api/engrams/{id}
Get a specific engram by ID
POST /api/engrams
Create a new engram
DELETE /api/engrams/{id}
Delete an engram by ID
GET /api/connections
List all connections
POST /api/search
Search for engrams
